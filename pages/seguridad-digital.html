🧠 ¿Qué es la seguridad digital?

La seguridad digital es una parte importante de la ciudadanía digital. Significa aprender a cuidarnos cuando usamos Internet, celulares, tablets o computadoras.

Internet nos ofrece muchas oportunidades, pero también puede presentar riesgos si no lo usamos con cuidado.

🌍 ¿Por qué es importante?

  • ✔️ Proteger los datos personales
  • ✔️ Evitar engaños o riesgos
  • ✔️ Mantener cuentas seguras
  • ✔️ Usar Internet con confianza

⚠️ Posibles riesgos

  • 🔴 Mensajes de desconocidos
  • 🔴 Compartir información personal
  • 🔴 Contraseñas débiles
  • 🔴 Archivos peligrosos
  • 🔴 Ciberacoso
  • 🔴 Páginas no seguras

🔒 Cómo protegerte

Datos personales

  • Nombre completo
  • Dirección
  • Teléfono
  • Escuela
  • Contraseñas
  • Fotos personales

Contraseñas seguras

✔ Letras + números + símbolos

✔ Ejemplo: M1Cl@v3Segur@

Descargas seguras

  • ✔ Sitios confiables
  • ✔ No abrir archivos desconocidos
  • ✔ Preguntar a un adulto

📱 Uso responsable

  • ✔ Bloquear pantalla
  • ✔ Actualizar dispositivos
  • ✔ Usar antivirus

Comportamiento en línea

  • ✔ Respetar a otros
  • ✔ No insultar
  • ✔ No difundir rumores

Reglas básicas

  • ✔ No compartir datos personales
  • ✔ Pensar antes de publicar
  • ✔ No aceptar desconocidos

🎮 Actividad: Verdadero o Falso

Responde las siguientes preguntas:

1. La seguridad digital ayuda a proteger tu información.

2. Es seguro compartir tu contraseña.

3. Debo descargar solo archivos de sitios confiables.

4. Internet nunca tiene riesgos.

5. Debo pensar antes de hacer clic o publicar.
